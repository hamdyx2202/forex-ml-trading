===========================================
     نظام التداول الآلي بالتعلم الآلي
         Forex ML Trading Bot
===========================================

## ملخص المشروع:
نظام تداول آلي متطور يستخدم الذكاء الاصطناعي للتداول في سوق الفوركس.
يتعلم من التاريخ ويتحسن باستمرار من كل صفقة.

## المميزات الرئيسية:
1. تعلم آلي متقدم (LightGBM + XGBoost)
2. يتعلم من 3 سنوات بيانات تاريخية
3. يتحسن تلقائياً كل يوم
4. إدارة مخاطر ذكية (1% لكل صفقة)
5. يعمل 24/7 على VPS
6. تنبيهات Telegram
7. لوحة تحكم ويب

## كيف يعمل النظام؟
1. يجمع البيانات من MetaTrader 5
2. يحلل 50+ مؤشر فني
3. يبحث عن أنماط ناجحة تعلمها
4. يدخل صفقات عالية الجودة فقط
5. يتعلم من كل صفقة ويتحسن

## متطلبات التشغيل:
- Hostinger VPS (خطة VPS 2 كافية)
- حساب MetaTrader 5 (تجريبي أو حقيقي)
- Python 3.9+
- 4GB RAM

## الملفات الأساسية:
/config/           - الإعدادات
/src/              - الكود الأساسي
/data/             - البيانات والنماذج
/logs/             - السجلات
main.py            - نقطة البداية
dashboard.py       - لوحة التحكم

## خطوات التشغيل السريعة:
1. اشترِ Hostinger VPS
2. انسخ الأوامر من SIMPLE_HOSTINGER_GUIDE.md
3. عدّل .env بمعلومات MT5
4. شغّل: python main.py trade

## الصيانة:
- يومياً: python run_daily_analysis.py
- أسبوعياً: python main.py collect
- شهرياً: python train_models.py
